Чтобы создать фон, который будет напоминать черный космос с редкими белыми звёздочками, можно использовать CSS. Вот пример кода, который вам поможет это реализовать: ```html Космический фон ``` ### Объяснение кода: 1. **HTML**: Создаём базовую структуру страницы. 2. **CSS**: - Устанавливаем черный фон страницы. - Создаём класс `.star` для звёзд, задавая им белый цвет, круглую форму и полупрозрачность. - Используем анимацию `twinkle` для мерцания звёзд. 3. **JavaScript**: - Генерируем случайное количество звёзд с заданными размерами и позициями на экране. - Каждая звезда добавляется к `body` и получает случайную анимацию мерцания. Вы можете изменить количество звёзд, их размеры и скорость анимации для создания вашего уникального космического фона!
